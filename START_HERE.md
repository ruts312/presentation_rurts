# Адам укуктары - Интерактивная Презентация 🎤

## ⚡ Самый быстрый старт (4 шага)

### 1️⃣ Установите все зависимости (ОДИН РАЗ)
Дважды кликните на:
```
install_dependencies.bat
```
Это установит все необходимые пакеты для Python и Node.js

### 2️⃣ Создайте файл с API ключом
```bash
cd backend
copy .env.example .env
notepad .env
```
Вставьте ваш OpenAI ключ:
```
OPENAI_API_KEY=sk-ваш-ключ-здесь
```

### 3️⃣ Запустите приложение
Дважды кликните на:
```
start_all.bat
```

### 4️⃣ Откройте в браузере
```
http://localhost:5173
```
